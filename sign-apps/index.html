Hướng dẫn sử dụng

  1. Cần tài khoản Apple Developer ($99/năm)
  2. Tạo Certificate trong Apple Developer Portal
  3. Download file .cer và import vào Keychain Access trên Mac
  4. Export thành file .p12 từ Keychain Access
  5. Download mobileprovision từ Apple Developer Portal

iOS yêu cầu HTTPS để download app OTA (Over-The-Air). Bạn có thể sử dụng Cloudflare Free SSL hoặc Let's Encrypt.
